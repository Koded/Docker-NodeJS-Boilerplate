# NodeJS Docker Container

## Setup

Create `config.yaml` in root of project:

```
docker_api:
  host: <DOCKER REST API HOST>
  port: <DOCKER REST API PORT

deploy:
  host: <HOST SERVER>
```




```
$ sudo npm install -g grunt-cli
$ npm install
```


## Development

App development in `./app`

## Deploying the app

```
$ grunt deploy-app
```
